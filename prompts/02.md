接下來，我想要強化改作業的工具
## 技術：
1. 增加 Google 試算表的整合
2. 可以將作業批改的結果，儲存於 Google 試算表中
3. 可以將考卷解答、Gemini API Key、Google 試算表的連結儲存於 LocalStorage 中

## 用戶的操作邏輯會調整為：
### 第一次使用
1. 需要輸入 Gemini API Key
2. 需要輸入 Google 試算表的連結
3. 登入 Google，用來直接授權調整 Google 試算表所使用

### 上傳解答階段
1. 上傳解答，並提供預覽讓老師檢視（Markdown 格式）
2. 確認後，將答案儲存於 LocalStorage 中

### 批改作業階段
1. 選擇試算表中的哪一個工作表
2. 需要輸入學生的座號（本工具可以大量批改，所以每一個座號代表一個學生）
3. 上傳單一學生的作業（可以拍照、上傳的形式），可以多張上傳
4. Gemini 3 Pro 進行作業批改
5. 僅產出批改結果，使用格式，並包含以下欄位（需使用 Prompt 明確定義）：
    - 學生姓名(如果沒有提供姓名，則不需要)
    - 學生座號(如果沒有提供座號，則不需要)
    - 學生作業批改結果（使用 Markdown 格式）
    - 學生的成績
    - 給予學生的回饋
6. 老師可以修改個別的學生批改結果
7. 下方提供按鈕，按下後將批改結果儲存於 Google 試算表中
